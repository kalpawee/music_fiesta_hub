## Getting Started

### Prerequisites

- Node.js >= 14.x (if using Node)
- npm or yarn
- Database server (if required)

### Installation

```bash
git clone https://github.com/kalpawee/music_fiesta_hub.git
cd music_fiesta_hub
npm install
```

### Running Locally

```bash
npm start
```
